# 代理充值结算比例调整 Impl

## 任务概述

根据已确认范围，将用户在线按量充值中的代理结算比例从 `1:10` 调整为 `1:7`：

- 余额充值：代理结算倍率改为 `7`。
- 图片积分充值：代理结算倍率同步改为 `7`。
- 套餐购买：不调整，仍按套餐售价减代理拿货价计算返现。

本次只影响新创建的充值订单。历史订单保留原 `agent_settlement_rate` 与 `agent_income_cny` 快照，不做重算。

## 核心代码说明

本次没有改动 `PaymentService._calculate_amounts()` 公式，只调整其读取的配置值。

余额充值公式保持：

```text
credited_usd = amount_cny * RECHARGE_USER_CNY_TO_USD_RATE
agent_cost_cny = credited_usd / RECHARGE_AGENT_CNY_TO_USD_SETTLEMENT_RATE
agent_income_cny = amount_cny - agent_cost_cny
```

图片积分充值公式保持：

```text
credited_image_credits = amount_cny * RECHARGE_IMAGE_CREDIT_USER_CNY_RATE
agent_cost_cny = credited_image_credits / RECHARGE_IMAGE_CREDIT_AGENT_CNY_RATE
agent_income_cny = amount_cny - agent_cost_cny
```

套餐购买逻辑保持：

```text
subscription_agent_rebate_cny = sale_price_cny - agent_cost_price_cny
```

套餐返现仍生成 `agent_subscription_sale_record`，不写入 `agent_cash_balance`。

## 测试验证

### 函数级公式验证

在 `backend` 目录执行：

```bash
python - <<'PY'
from decimal import Decimal
from app.services.payment_service import PaymentService
from app.config import settings

cases = [
    ('platform_balance', Decimal('100'), None, 'balance'),
    ('agent_balance', Decimal('100'), 1, 'balance'),
    ('agent_image_credit', Decimal('100'), 1, 'image_credit'),
]
print('balance_agent_rate=', settings.RECHARGE_AGENT_CNY_TO_USD_SETTLEMENT_RATE)
print('image_agent_rate=', settings.RECHARGE_IMAGE_CREDIT_AGENT_CNY_RATE)
for name, amount, agent_id, recharge_type in cases:
    credited_usd, credited_image_credits, agent_rate, agent_income_cny = PaymentService._calculate_amounts(amount, agent_id, recharge_type)
    print(name, 'credited_usd=', credited_usd, 'credited_image_credits=', credited_image_credits, 'agent_rate=', agent_rate, 'agent_income_cny=', agent_income_cny)
PY
```

验证结果：

```text
balance_agent_rate= 7
image_agent_rate= 7
platform_balance credited_usd= 500.000000 credited_image_credits= 0.000 agent_rate= 7 agent_income_cny= 0.00
agent_balance credited_usd= 500.000000 credited_image_credits= 0.000 agent_rate= 7 agent_income_cny= 28.57
agent_image_credit credited_usd= 0.000000 credited_image_credits= 500.000 agent_rate= 7 agent_income_cny= 28.57
```

结论：

- 直营网余额充值 100 元：代理分润 `0`。
- 代理用户余额充值 100 元：用户到账 `500`，代理分润 `28.57`。
- 代理用户图片积分充值 100 元：用户到账 `500` 积分，代理分润 `28.57`。
