# 代理套餐在线充值与套餐排序 Impl

## 任务概述

- 在代理系统配置中新增站点级“套餐在线充值”开关。
- 在用户充值页根据站点开关显示或隐藏套餐购买入口，并在后端创建套餐订单时强校验。
- 在管理员套餐模板管理中新增排序控制，影响 `user/recharge` 套餐展示顺序。

## 核心代码说明

- `agent.subscription_online_recharge_enabled` 默认值为 `1`，保持已有代理站点升级后默认可购买套餐。
- `AgentService.is_subscription_online_recharge_enabled` 先继承总在线充值开关，再判断套餐独立开关。
- `/api/user/payment/subscription-plans` 在套餐在线充值关闭时返回空列表。
- `PaymentService.create_recharge_order` 对 `recharge_type=subscription` 增加后端拒绝逻辑，错误码为 `AGENT_SUBSCRIPTION_ONLINE_RECHARGE_DISABLED`。
- 管理端套餐模板使用已有 `sort_order` 字段；后端列表和用户可购套餐列表均已按 `sort_order ASC, id ASC` 返回。
