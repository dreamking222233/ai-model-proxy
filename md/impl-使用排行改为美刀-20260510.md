# 使用排行改为美刀

## 任务概述
- 将 `admin/ranking` 与 `user/ranking` 的排行口径从 Token 消耗改为美元消费。
- 保持现有排行入口与交互不变，仅切换后端聚合指标和前端展示文案。

## 文件变更清单
- `backend/app/services/log_service.py`
- `frontend/src/views/user/UsageRanking.vue`

## 核心代码说明
1. 后端排行聚合：
   - 在排行查询中增加 `ConsumptionRecord.total_cost` 汇总。
   - 排序主键由 `RequestLog.total_tokens` 改为 `ConsumptionRecord.total_cost`。
   - 返回结果增加 `total_cost` 字段，保留请求次数、Token、最近使用时间等辅助字段。
2. 前端排行展示：
   - 英文单位从 `Tokens` 改为 `USD`。
   - 领奖台和列表主数值从 `total_tokens` 切换为 `total_cost`。
   - 页面副标题改为“按美元消费实时更新”。

## 测试验证
- `python -m py_compile backend/app/services/log_service.py`
- `npm run build`

## 备注
- `admin/ranking` 复用 `frontend/src/views/user/UsageRanking.vue`，因此本次页面改动会同时作用于管理端与用户端。
