# 媒体工作台缓存 Impl

## 任务概述

为 `user/media-workbench` 增加浏览器本地缓存：生成图片或视频成功后保存到用户浏览器，页面刷新后自动恢复最近结果。

## 文件变更清单

- `frontend/src/utils/mediaWorkbenchStorage.js`
  - 新增媒体工作台缓存工具。
  - `localStorage` 保存结果索引。
  - `IndexedDB` 保存图片 dataUrl 与视频 Blob。
  - 按用户 ID 隔离，最多保留 20 条，缓存有效期 7 天。
- `frontend/src/views/user/MediaWorkbench.vue`
  - 页面初始化时加载缓存结果。
  - 图片生成成功后保存图片结果。
  - 视频生成成功后保存视频 Blob。
  - 清空结果时同步删除本地缓存。

## 核心代码说明

- 图片结果仍直接使用 data URL 展示，并将 data URL 写入 IndexedDB。
- 视频结果从 `/v1/videos/{id}/content` 拉取 Blob，创建 object URL 展示，同时保存 Blob 到 IndexedDB。
- 刷新页面后根据 localStorage 索引从 IndexedDB 恢复图片/视频。
