## 测试结果

### ChatGPT Image Compatible

直连测试路径：

- `POST /v1/images/generations`
- `POST /v1/image/created`
- `POST /images/generations`

结果：均未成功返回，约 `129.5s` 后报 `RemoteProtocolError: Server disconnected without sending a response`。此前单路径测试也出现 `ReadTimeout`。

结论：该渠道当前不可用，和正式库 `enabled=0/is_healthy=0` 一致。已将本地该渠道恢复为禁用/异常，避免 1K 请求卡在异常 compatible 渠道。

### 首尔-gpt-image-2 Native Size

- 直连 `/v1/images/generations`：成功。
- 本地代理 `/v1/images/generations`：
  - `model=gpt-image-2`
  - `image_size=2K`
  - `aspect_ratio=16:9`
  - 命中 `channel_id=24 首尔-gpt-image-2`
  - 返回成功，`request_id=4739e598-f6fd-44ce-9b94-9b1529c9b5a2`
  - 日志扣费：`image_credits_charged=0.700`
  - 用户媒体积分：`200.000 -> 199.300`

### 首尔-Grok

- 直连 JSON 请求返回 `400 Field required`，确认上游需要 multipart 表单。
- 直连 multipart `/v1/videos`：成功返回 `queued`。
- 本地代理 `/v1/videos`：
  - `model=grok-video`
  - 命中 `channel_id=25 首尔-Grok`
  - 返回成功，`status=queued`
  - `image_credits_charged=0`
  - `billing_status=pending_completion`

结论：视频创建链路正常，创建阶段不扣费，等待完成/内容校验后才扣费的逻辑保持正确。
