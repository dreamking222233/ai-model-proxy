# 用户-模型展示 Impl

## 任务概述

修复 `user/models` 页面模型卡片缺少价格和上下文信息的问题。根据历史提交恢复卡片价格区和标签区，并兼容当前版本新增的按请求计费模型。

## 文件变更清单

- `frontend/src/views/user/ModelList.vue`
  - 恢复模型卡片顶部信息、中部计费、底部标签结构。
  - Token 计费展示 Input/Output 每百万 token 价格。
  - 媒体积分计费展示图片/视频消耗。
  - 请求次数计费展示单次请求固定价格。
  - 免费模型展示“免费使用”，不再显示 `$0 /1M` 的 Token 价格结构。
  - 恢复 `max_tokens` 上下文标签展示。
  - 恢复完整卡片样式和响应式网格，并补充 480px 以下移动端布局。
- `md/plan-用户-模型展示-20260608.md`
  - 记录本次修复方案。

## 核心代码说明

- `billing_type === 'request'` 时通过 `getRequestPriceText(model)` 格式化 `$x / 次`。
- `billing_type === 'image_credit'` 继续复用 `getImageCreditText(model)` 展示媒体积分规则。
- 普通 Token 计费模型展示 `input_price` 与 `output_price`。
- 非图片/视频模型在存在 `max_tokens` 时展示 `Context` 标签。

- `npm run build`：通过，存在历史资源体积 warning，不影响本次修复。
