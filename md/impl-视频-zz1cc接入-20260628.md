# impl-视频-zz1cc接入-20260628

## 任务概述

根据 `md/plan-视频-zz1cc接入-20260628.md` Plan v2，为当前系统新增 zz1cc 视频渠道接入，支持 `video-ds-2.0` 和 `video-ds-2.0-fast`。用户侧继续沿用现有视频接口和返回结构：

- `POST /v1/videos`
- `POST /v1/created/video`
- `GET /v1/videos/{video_id}`
- `GET /v1/videos/{video_id}/content`

首版不支持 `/v1/chat/completions` 视频兼容入口调用 zz1cc 视频模型，会返回 `VIDEO_MODEL_NOT_SUPPORTED`。

## 核心代码说明

- zz1cc 创建请求走 JSON：

```json
{
  "model": "video-ds-2.0-fast",
  "prompt": "...",
  "seconds": 15,
  "aspect_ratio": "9:16",
  "images": ["data:image/png;base64,..."]
}
```

- 秒数规则：`None` 或空值默认 `15`，非 `15` 直接返回 `INVALID_VIDEO_SECONDS`。
- 任务 ID 规则：`id = body.id || body.task_id || body.request_id`，本地 route 和对外响应统一使用归一后的 `id`。
- 查询状态规则：`completed/succeeded/success/done -> completed`，`pending/queued -> queued`，`in_progress/processing/running -> in_progress`，`failed/error/cancelled/canceled/expired -> failed`。
- content 获取：zz1cc 直接使用 `/v1/videos/{id}/content`，并复用现有视频内容校验和完成后计费逻辑。
