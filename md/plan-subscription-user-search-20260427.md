# 套餐管理用户搜索计划

## 用户原始需求

优化管理端和代理端套餐管理逻辑，支持搜索用户名称并下拉选择用户；代理端只能选择自己的用户。

## 技术方案

1. 管理端用户列表接口补充角色过滤参数，支持只查询终端用户。
2. 管理端套餐管理页将“用户 ID 输入”改为远程搜索下拉。
3. 代理端套餐管理页将“用户 ID 输入”改为远程搜索下拉。
4. 代理端继续复用现有代理用户接口，天然限制为当前代理名下终端用户。
5. 对从用户页跳转附带 `user_id` 的场景做预载处理。

## 涉及文件

- `backend/app/api/admin/user.py`
- `frontend/src/views/admin/SubscriptionManage.vue`
- `frontend/src/views/agent/SubscriptionManage.vue`

## 实施步骤

1. 管理端用户列表接口新增 `roles` 参数。
2. 管理端套餐页接入远程用户搜索选择。
3. 代理端套餐页接入远程用户搜索选择。
4. 执行后端语法检查和前端构建验证。
