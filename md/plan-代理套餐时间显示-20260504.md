# 代理套餐时间显示修复 Plan

## 用户原始需求

`agent/subscription` 页面开通套餐后，当前实际时间如 `2026-05-04 00:07`，页面却显示 `2026-05-04 08:06:37`。同时检查 `admin/subscription` 页面使用的时间口径。

## 技术方案设计

1. 检查 `agent/subscription` 前端时间格式化方式。
2. 检查 `admin/subscription` 前端时间格式化方式。
3. 检查后端 `SubscriptionService` 创建套餐时间与序列化字段。
4. 修复 `agent/subscription` 使用错误的 UTC 格式化函数。
5. 记录 admin 页当前时间口径，避免后续误判。

## 涉及文件清单

- `frontend/src/views/agent/SubscriptionManage.vue`
- `md/plan-代理套餐时间显示-20260504.md`
- `md/impl-代理套餐时间显示-20260504.md`

## 实施步骤概要

1. 确认 `SubscriptionService.get_current_time()` 返回北京时间无时区时间。
2. 确认 `admin/subscription` 使用 `formatDate()`，按本地时间直接展示。
3. 将 `agent/subscription` 从 `formatUtcDate()` 改为 `formatDate()`。
4. 运行前端构建验证。
