# 去除联系方式门槛实施计划

## 用户原始需求

移除平台直营客户 QQ、微信联系方式的充值门槛，恢复所有直营客户直接查看联系方式；代理端逻辑保持不变，并将最新代码和数据库升级同步到服务器。

## 技术方案设计

1. 删除用户资料接口中按支付订单累计金额隐藏联系方式的逻辑，恢复站点配置原始联系方式。
2. 删除公开站点配置接口中对平台 QQ、微信的脱敏，避免未登录和登录接口返回不一致。
3. 移除管理端公告配置页和接口中的门槛字段，保留公告标题、内容、微信、QQ 配置。
4. 移除直营用户首页、充值页和公告展示中的门槛提示与锁定状态，直接展示已配置联系方式。
5. 从初始化 SQL 中移除默认门槛配置，并提供删除已有配置的幂等升级 SQL。
6. 执行本地和生产数据库升级，构建前端、重启后端并验证健康状态。
