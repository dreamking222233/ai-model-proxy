# 媒体工作台缓存 Review

## 审查结论

实现符合需求，未发现阻塞性问题。

- 使用 `localStorage` 保存结果索引、`IndexedDB` 保存媒体内容，避免把大体积视频 Blob 直接写入 localStorage。
- 缓存按用户 ID 隔离，匿名用户使用独立命名空间。
- 页面初始化会恢复最近缓存结果；图片、视频生成成功后会写入缓存。
- 清空结果会同步清理缓存索引和 IndexedDB 媒体资产。
- 缓存限制为最多 20 条、7 天有效期，避免无限增长。

cd frontend

npm run build
